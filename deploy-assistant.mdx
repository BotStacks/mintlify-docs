---
title: "Deploy web assistant"
description: "Deploy your web assistant in less than a minute!"
---

<Note>
  If you haven't yet created your AI agent, check out our [Assistant designer
  documentation](/designer/introduction) to get started!
</Note>

Once you've created a web assistant with our designer, you can quickly deploy that
assistant onto your website or landing page.

<Steps>
  <Step title="Choose your assistant">
    Go to the dashboard and open the extra options for the desired agent. Select
    "Customize web assistant". ![Create new intent
    button](/images/assistant-options.png)
  </Step>
  <Step title="Get your code snippet">
    Customize the web assistant as you desire and click "Copy" to copy the code
    snippet to your clipboard. ![Customize web
    assistant](/images/customize-web-assistant.png)
  </Step>
  <Step title="Paste your code snippet to your website">
    Paste the code snippet at the end of your html `<body>` tag.
    ![Example website](/images/assistant-code.png)
  </Step>
</Steps>
