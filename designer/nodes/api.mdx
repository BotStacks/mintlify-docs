The API node provides a way to call an external RESTful API to retrieve data or execute a function. For example, you
can use this to perform a a POST request to create a booking with a third party service.
![API Node](/images/api-node.png)

### Method

The API node supports the following request methods:

- GET
- POST
- PUT
- PATCH
- DELETE

### URL

This is the endpoint of your external service.

### Headers

You can add as many headers as needed to fulfill your request

### Body

This is shown when a POST, PUT, or PATCH request method is selected. This is in JSON format, so make sure you format
correctly.
