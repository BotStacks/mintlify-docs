The text input node is used to add specific input into the conversation.

<img 
height="500"  
src="/images/node-text.png"
/>

A text input node can be used anywhere within the conversation flow. It requires you to specify the value you want to output, which could include text or references to other input nodes.

<Tip>You can also reference input nodes such as `{llm-0}` on the Text Node.</Tip>
