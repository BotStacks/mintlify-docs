This SDK integrates a fully serviced chat experience on the [BotStacks](https://botstacks.ai/) platform.


<Warning>
  The android SDK is still in active development. Check in regularly for
  updates!
</Warning>


## Installation
<img 
src="https://img.shields.io/maven-central/v/ai.botstacks/chat-sdk"
/>

### Compose Multiplatform

Add `ai.botstacks:chat-sdk:{version}` to your dependencies

```Gradle
val commonMain by getting {
    dependencies {
        [...]
+       implementation("ai.botstacks:chat-sdk:{version}")
        [...]
    }
}
```
<Note> To setup Compose Multiplatform click [here](https://docs.botstacks.ai/sdk-reference/android/compose-setup)</Note>

### Android Only

If you are only targeting Android the dependency is:


```Gradle
dependencies {
    [...]
+   implementation("ai.botstacks:chat-sdk-android:{version}")
    [...]
}
```
<Note> To setup android click [here](https://docs.botstacks.ai/sdk-reference/android/setup)</Note>
