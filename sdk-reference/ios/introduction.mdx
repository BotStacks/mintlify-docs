## Overview

The iOS UIKit is a collection of Swift components crafted to help engineers build a
feature-rich chat experience.

## Installation

### Cocoapods

This iOS SDK is accessible via Cocoapod as `BotStacksChat`. Simple add to your podfile

```bash
use_frameworks!

target 'MyApp' do
  pod 'BotStacksChat'
end
```

### Swift package manager

```Swift
dependencies: [
    .package(url: "https://github.com/BotStacks/ios-sdk.git", .upToNextMajor(from: "1.0.1"))
]
```
