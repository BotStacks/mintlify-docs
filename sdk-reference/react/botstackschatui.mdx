## Component

The `BotStacksChatUI` component provides a quick and easy full page chat application. It has all the available features
and is a great way to see what the UI Kit and SDK are capable of.

It's what our demo site uses. [Check it out!](https://demo.botstacks.ai)

## API

### Import

```js
import { BotStacksChatUI } from "@botstacks/chat-react";
```
