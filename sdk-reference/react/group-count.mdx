---
title: "GroupCount"
description: "A badge for group display."
---

<Note>This component is NOT data aware</Note>

## Component

The GroupCount component displays a grouped avatar image with a count of the number of members.

## API

### Import

```js
import { GroupCount } from "@botstacks/chat-react";
```

### Props
