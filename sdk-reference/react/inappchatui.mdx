## Component

The `InAppChatUI` component provides a quick and easy full page chat application. It has all the available features
and is a great way to see what the UIKit and SDK are capable of.

It's what our demo site uses. [Check it out!](https://inappchat.io)

## API

### Import

```js
import { InAppChatUI } from "@inappchat/react";
```
