This component renders a given list of `Chat` channels in a `Column`, each chat in its own ChannelRow, with the `Chat.displayName` as the title and the subtitle showing a preview of the members with in it. This is utilized in the [UserDetailsView](https://docs.botstacks.ai/sdk-reference/android/views#userdetailsview)  to show channels the current user has in common with any other user.

```Kotlin
ChannelGroup(
    channels = listOf(yourChannels),
)
```
